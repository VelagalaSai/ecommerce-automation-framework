# E-Commerce Automation Testing Framework

A complete Selenium test automation framework built using Python, pytest, and Page Object Model (POM) design pattern. Tests are executed against the [SauceDemo](https://www.saucedemo.com) e-commerce demo application.

---

## Tech Stack

- **Language:** Python 3.x
- **Automation Tool:** Selenium WebDriver 4.x
- **Test Framework:** pytest
- **Design Pattern:** Page Object Model (POM)
- **Browser:** Microsoft Edge
- **Reporting:** pytest-html
